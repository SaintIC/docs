About

Flask-PluginKit: Web program plugin development kit based on flask.

With the Flask-PluginKit extension, you can add custom plugins or use third-party plugins for Web applications without changing the core code, and extend the application in a variety of ways.

Third-party plug-ins will also be displayed in the plug-in market, please look forward to it.
